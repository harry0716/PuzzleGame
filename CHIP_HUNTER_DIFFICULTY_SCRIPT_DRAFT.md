# Chip Hunter Independent Difficulty Script Draft

## Purpose

This draft converts the `chip-hunter` difficulty spec into a writing-ready
three-script structure.

It is the next step before live implementation in `scene-registry.js`.

The scene keeps one shared identity:

- scene id: `chip-hunter`
- theme: AI devices, semiconductors, electronics, AIoT, and system integration

But each difficulty tier should now feel like a different visit or challenge
path.

## Shared Role Family

Recommended shared result roles:

1. `AI 模型探索者`
2. `AIoT 系統整合師`
3. `智慧製造指揮員`
4. `實作創新挑戰者`

Recommended shared traits:

- `ai`
- `system`
- `industry`
- `maker`

## Easy Script

### Positioning

This version is the introductory visit path.

It should help first-time visitors quickly recognize what the AI lab is trying
to show, why certain devices matter, and how electronics and AI connect.

### Landing Direction

Suggested tone:

- welcoming
- guided
- recognition-focused

Suggested landing copy direction:

- this version helps you quickly understand what you are seeing in the AI lab
- you will focus on core devices, simple role connections, and visible examples
- it is suitable for first-time visitors and lower-pressure activity use

### Question Set

#### Q1. Which Device Most Represents AI Computing Power?

Type:

- `single-choice`

Prompt direction:

- when students first enter the lab, which piece of equipment most clearly
  represents "real AI computing power"?

Correct concept:

- GPU / AI server hardware is the strongest symbol of high-level AI computing

Primary trait:

- `ai`

#### Q2. What Makes This Department Different?

Type:

- `single-choice`

Prompt direction:

- why does this scene want students to understand more than only "electronics
  hardware"?

Correct concept:

- the scene is about electronics + AI + control + system integration together

Primary trait:

- `system`

#### Q3. Fast Recognition Under Time Pressure

Type:

- `timed-choice`

Prompt direction:

- if you only have a few seconds to identify the most AI-related exhibit, what
  should you look for first?

Correct concept:

- find the device that most directly represents computation or model training

Primary trait:

- `ai`

#### Q4. Visit Understanding Order

Type:

- `ordering`

Prompt direction:

- put the visit-understanding flow in a logical order: see the equipment,
  understand the purpose, connect it to applications, then reflect on your
  interest

Correct concept:

- guided understanding should move from visible object to meaningful
  interpretation

Primary trait:

- `industry`

#### Q5. Where Would You Look First?

Type:

- `branching`

Prompt direction:

- if you stand in the exhibit area for the first time, would you rather begin
  with the AI server area or the system-application area?

Correct educational direction:

- both are valid, but the scene should help the player understand the difference
  between hardware-first and application-first exploration

Primary trait:

- `system`

#### Q6. Recognize The AI Core Image

Type:

- `image-choice`

Prompt direction:

- which picture most closely matches the AI computing core of this scene?

Correct concept:

- the AI server rack is the clearest image-level representation of the scene's
  core computing identity

Primary trait:

- `ai`

#### Q7. Match The Main Themes

Type:

- `matching`

Prompt direction:

- match AI server, Factory IO, and AIoT with their most direct meanings

Correct concept:

- students should be able to distinguish compute, smart factory simulation, and
  integrated systems at a simple level

Primary trait:

- `system`

### Easy Result-Copy Direction

- more encouraging
- more directional
- less technical density

## Hard Script

### Positioning

This version is the advanced interpretation script.

It should not just ask "what device is this?" but "why does this scene matter as
an AI + electronics ecosystem?"

### Landing Direction

Suggested tone:

- sharper
- comparative
- systems-oriented

Suggested landing copy direction:

- this version asks you to interpret the scene at a deeper level
- you will compare devices, systems, application layers, and field-of-study
  directions under a tighter challenge flow
- it is suitable for competition or advanced visitors

### Question Set

#### Q1. Symbol Or Substance?

Type:

- `single-choice`

Prompt direction:

- among several visible lab items, which one best symbolizes actual AI
  computing power rather than just general technology presence?

Correct concept:

- high-performance AI compute hardware should be distinguished from ordinary
  presentation or office equipment

Primary trait:

- `ai`

#### Q2. Which Learning Path Best Matches This Scene?

- `single-choice`

Prompt direction:

- what makes this scene stronger than a narrow "only hardware" or "only AI
  software" interpretation?

Correct concept:

- `system`

Primary trait:

- `ai`

Correct concept:

Primary trait:

- `ai`

#### Q7. Match Technology Layer To Sector Value

Type:

- `matching`

Prompt direction:

Correct concept:

- the scene’s deeper point is not isolated devices, but how those layers support

- `industry`
