# Slide Notes

This file tracks the intended presenter flow for the activity.

## Suggested 10-Slide Story Arc

1. Welcome and event theme
2. What is AI in daily life
3. What happens inside an AI lab
4. AI + hardware + systems integration
5. GPU and compute concepts
6. AIoT and applied engineering
7. Factory / industrial simulation examples
8. Career or skill-path mapping
9. Invite students to join the quiz
10. Show QR code and start the activity

## Presenter Workflow

Use the following pages together:

- `presenter.html` on the main display
- `index.html` for participant devices
- `leaderboard.html` for manual ranking checks

## On-Site Checklist

- confirm Wi-Fi or mobile network is available
- confirm `config.js` points to the correct event
- open `presenter.html`
- confirm QR code loads correctly
- verify leaderboard refreshes
- test one sample submission before the audience starts
